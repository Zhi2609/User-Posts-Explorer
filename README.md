# User Posts Explorer 🚀

Una aplicación web sencilla que conecta con una API externa para visualizar publicaciones de usuarios de forma dinámica.

## ✨ ¿Qué hace este proyecto?
* **Carga de Usuarios:** Al presionar un botón, obtiene una lista de usuarios desde una API.
* **Filtro Dinámico:** Al seleccionar un usuario del menú, se buscan y muestran sus publicaciones específicas.
* **Diseño con Bootstrap:** Utiliza tarjetas (Cards) de Bootstrap para mostrar el título y contenido de cada post.

## 🛠️ Tecnologías
* **HTML5 & Bootstrap 5:** Para la estructura y el diseño rápido.
* **JavaScript (Fetch API):** Para consumir datos de `jsonplaceholder.typicode.com`.
